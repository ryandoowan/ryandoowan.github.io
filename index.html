Projects

Thread Manager

Developed a thread manager for handling the creation, management, and execution of threads on a multi-core system in C++ 17. The thread library includes features for conditional variables, mutexes, and generic threads. In addition, it supported inner process interrupts and various thread lifetime management operations like yield and join.

Instagram App Replica

Deployed an interactive website with AWS that mirrored Instagram by using Python 3 for server-side operations, JavaScript for dynamic client-side features, and SQL for database management. Leveraging ReactJS, crafted responsive interfaces and integrated REST APIs for data exchange, optimizing performance with AJAX calls to REST APIs.

Google Minesweeper Solver

Developed an algorithm in Python 3 to solve Google Minesweeper, leveraging packages such as PyAutoGUI and screen capture tools. The algorithm automates gameplay by analyzing the game board, identifying patterns, and making strategic moves to efficiently solve the puzzle.

Neural Network multiclass classifier

Developed a Python algorithm to train a 7-layer Convolutional Neural Network (CNN) with 2 pooling layers. The model was trained on a dataset of 2000 images across 10 different dog breeds and validated on 500 images, achieving an accuracy of 90.7%. Training data included different types of data augmentations, such as greyscale images and augmented images (distorted/rotated) to enhance model robustness and accuracy.
